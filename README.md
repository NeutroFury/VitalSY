# VitalSY Mobile 📱💉

**VitalSY** es una plataforma tecnológica integral diseñada para optimizar la gestión de pacientes con **Diabetes Tipo 1**. Este repositorio contiene el módulo móvil, enfocado en superar la fragmentación de las herramientas actuales mediante una interfaz centralizada y proactiva.

## 🚀 Propósito del Proyecto
La solución busca centralizar el **monitoreo de glucosa**, el **registro de alimentación** y el **cálculo de dosis de insulina**. Gracias a la integración de **Inteligencia Artificial predictiva** (Machine Learning y Deep Learning), la aplicación es capaz de anticipar riesgos de salud en tiempo real, permitiendo una toma de decisiones informada y segura.

## ✨ Características Principales
* **Monitoreo Centralizado:** Visualización de niveles de glucosa en tiempo real.
* **Gestión Nutricional:** Registro avanzado de alimentación para el conteo de carbohidratos.
* **Calculadora de Insulina:** Algoritmos para el cálculo preciso de dosis según ingesta y niveles actuales.
* **IA Predictiva:** Modelos de ML/DL diseñados para la anticipación de riesgos.
* **Sincronización:** Conectividad con sensores médicos externos para la captura automatizada de datos.
